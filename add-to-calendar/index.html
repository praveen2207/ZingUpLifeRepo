Introduction

This script will allow you to add events to either Apple's iCal/Calendar (Mac or iPhone/iPad), Outlook 2007 (.ics file) and Google Calendar.

Simply copy one of the examples below into your web page and adjust the title, and start time accordingly (these are the only two required parameters).

Make sure that addToCalendar.php is either in the same directory as your links or that you correctly adjust the relative path in the href section.

URL Parameters to Be Set

  • format - (required; "ical" or "google")
  • title - the title of your event (required)
  • start - the date/time that the event starts (required; YYYY-MM-DD+H:i:s format; the "+" since it will be passed in the URL)
  • finish (optional; can use "length" instead)
  • length (optional; defaults to $defaultEventLength [set in addToCalendar.php])
  • location - where the event is taking place (optional)
  • url - the URL of your Event Page (optional)
  • imgurl - the URL of the image attachment (optional; e.g. a map using Google's Static Map API; example below)
  • imgname - the name of the image file (defaults to staticmap.png; don't forget the .png extension!)
Make sure you urlencode all of the above variables since they will be passed in the URL.
